# Incident Playbooks

Projector failure
- Check AV Controller AC-2 before the electrical panel.
- Confirm HDMI/source input and controller power.
- Inspect Electrical Panel EP-2 breaker 14 only if authorized.

Network rack alert
- Inspect NR-1 status lights without unplugging live equipment.
- Check whether AV, ticket scanner, or staff Wi-Fi services are also degraded.
- Escalate to network support if multiple services are affected.

Power panel issue
- Do not bypass safety procedures.
- Breaker inspection is allowed only for authorized personnel.
- Stop if there is smell, smoke, heat, visible damage, or repeated trip.

Smoke detector alert
- Do not silence or bypass alarms.
- Move people away from the affected corridor.
- Escalate to authorized safety/fire personnel immediately.

Crowd flow issue
- Route staff through Security Desk.
- Keep First Aid accessible.
- Avoid blocking Gate 1, Gate 2, or Gate 3 Corridor.
