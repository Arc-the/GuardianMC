# Incident Playbooks

## Projector Failure

### Symptoms
- No image on screen
- Signal lost message
- Flickering display
- Color distortion

### Diagnostic Path
1. **Check AV Controller** (AV Booth)
   - Verify power LED is on
   - Check signal LED status
   - Confirm input source selection

2. **Verify Source Signal**
   - Check HDMI/source cable connection
   - Test with alternate input if available
   - Check source device status

3. **Check Projector Status** (Zone B ceiling)
   - Look for power indicator light
   - Listen for fan operation
   - Note any error indicators

4. **Electrical Check** (if authorized)
   - Inspect Electrical Panel EP-2
   - Check breaker 14 position
   - Look for signs of trip

### Escalation
- If breaker has tripped multiple times: stop and call electrician
- If burning smell or visible damage: evacuate area, call safety

## Network Rack Alert

### Symptoms
- AV system connectivity loss
- Ticket scanner failures
- Wi-Fi disruptions
- Camera feed interruptions

### Diagnostic Path
1. **Check Network Rack NR-1** (Network Room)
   - Verify power to rack
   - Check switch status lights
   - Look for red/amber indicators

2. **Temperature Check**
   - Verify cooling fans operating
   - Check room temperature
   - Look for blocked ventilation

3. **Port Inspection**
   - Identify affected port range
   - Check cable connections
   - Test with known working cable

### Escalation
- If multiple systems affected: contact IT immediately
- If overheating detected: power down non-critical systems

## Power Panel Issue

### Symptoms
- Multiple devices offline
- Lights flickering
- Breaker in tripped position

### Diagnostic Path
1. **Visual Inspection Only**
   - Check panel for damage
   - Look for burn marks
   - Note any unusual smell

2. **Document Status**
   - Which breakers are tripped
   - Which systems are affected
   - Time of occurrence

3. **Do NOT Reset**
   - Wait for authorized electrician
   - Keep area clear
   - Report to control room

### Escalation
- Always escalate power panel issues to qualified electrician
- Never attempt repairs without authorization

## Smoke Detector Alert

### IMMEDIATE ACTIONS
1. **Do NOT silence alarm**
2. **Alert security desk**
3. **Proceed to Gate 3 Corridor** (location of SD-G3)
4. **Visual check from safe distance**

### Assessment
- Is there visible smoke?
- Is there burning smell?
- Are people in the area safe?

### Response
- If smoke/fire confirmed: Begin evacuation, call emergency services
- If false alarm suspected: Wait for fire safety to verify
- Keep crowd calm and away from area

### NEVER
- Enter smoke-filled area
- Attempt to silence or reset detector
- Delay alerting safety personnel
- Tell others it's "probably nothing"

---

## Crowd Flow Issue

### Symptoms
- Density exceeding safe levels
- Blocked exits
- Crowd pushing/shoving
- People down

### Immediate Actions
1. **Alert security desk**
2. **Identify blocked exits**
3. **Open additional exits if authorized**
4. **Direct crowd to less dense areas**

### Escalation
- If injury suspected: Call first aid
- If crowd uncontrollable: Request security backup
- If exits blocked: Priority one emergency
