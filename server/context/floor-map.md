# Floor Map - Venue Layout

## Main Areas

### Main Hall
- Central gathering space
- Primary event viewing area
- Adjacent to Zone A and Zone B

### Zone A
- West viewing section
- Projector A overhead
- Capacity: 150 people
- Emergency exit to Gate 1

### Zone B
- East viewing section
- Projector B overhead
- Capacity: 150 people
- Emergency exit to Gate 2

## Support Rooms

### AV Booth
- Located between Zone A and Zone B
- Contains AV Controllers AC-1 and AC-2
- Direct line of sight to both projection screens
- Access: Technical staff only

### Network Room
- Behind AV Booth
- Contains Network Rack NR-1
- Climate controlled
- Access: IT staff only

### Electrical Room
- Adjacent to Network Room
- Contains Electrical Panels EP-1, EP-2, EP-3
- High voltage warning
- Access: Authorized electricians only

## Entry/Exit Points

### Gate 1
- West entrance
- Primary public entry
- Security checkpoint

### Gate 2
- East entrance
- Secondary public entry
- Security checkpoint

### Gate 3 Corridor
- North service corridor
- Staff access only
- Contains Smoke Detector SD-G3
- Emergency exit route

## Safety Stations

### First Aid
- Near Gate 1
- Staffed during events
- Medical supplies and AED

### Security Desk
- Main Hall entrance
- Monitors all camera feeds
- Emergency communication hub
