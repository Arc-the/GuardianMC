# Venue Assets

## Projector B
- **Zone:** Zone B
- **Location:** Ceiling mount above Zone B viewing area
- **Connected To:** AV Controller AC-2
- **Power Panel:** Electrical Panel EP-2
- **Breaker Number:** 14
- **Safe Checks:**
  1. Check AV Controller AC-2 power and status lights
  2. Verify HDMI input source is active
  3. Check power indicator on projector
  4. If authorized, inspect breaker 14 on EP-2

## AV Controller AC-2
- **Location:** AV Booth
- **Connected To:** Projector B, Main audio system
- **Network:** Connected via Network Rack NR-1
- **Status Indicators:** Power LED, Signal LED, Network LED
- **Notes:** Controls all Zone B audiovisual equipment

## Network Rack NR-1
- **Location:** Network Room
- **Supports:** AV system, ticket scanners, staff Wi-Fi, security cameras
- **Critical Ports:**
  - Port 1-4: Security systems
  - Port 5-8: AV equipment
  - Port 9-12: Staff devices
- **Cooling:** Active ventilation required

## Electrical Panel EP-2
- **Location:** Electrical Room
- **Supports:**
  - Projector B (Breaker 14)
  - Zone B lighting (Breakers 10-13)
  - AV Booth outlets (Breakers 15-16)
- **Access:** Authorized personnel only
- **Safety:** Check for visible damage before any inspection

## Smoke Detector SD-G3
- **Location:** Gate 3 Corridor
- **Type:** Photoelectric smoke detector
- **Connected To:** Central fire alarm panel
- **Safety Instruction:** Do NOT silence or bypass alarms. Verify area safety and escalate to fire/safety staff immediately.

- **Connected To:** AV Controller AC-1
